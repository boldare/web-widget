# BotMan Web Widget

## Before development start
Copy first default configuration for widget before build.
```
cp src/widget/config.ts.dist src/widget/config.ts 
```

## Building
Optionally you can export `EXPORT_BUILD_PATH` in terminal
```
export EXPORT_BUILD_PATH=path_to_destination
``` 
After exporting the variable builder also copy `build/` directory into exported destination.
